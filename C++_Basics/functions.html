Functions in C++

What is a function?

  • A function is an special section of a program that can act on data and return a value.
  • To call a function is to use it typing its name inside your program.
  • Inside a function you can call another functions.
  • Every C++ program has at least one function called Main()
  • The function Main() is mandatory in every C++ program

Declaring and defining a function

Before using a function you must tell the compiler what is your function about (high level description) that is, its name, what value it returns and what parameters it accepts. This is called declaration or prototyping.
